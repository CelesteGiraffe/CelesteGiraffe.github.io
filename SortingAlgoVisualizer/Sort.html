How It Works

  • Each button triggers the algorithm on the current dataset with animated swaps and comparisons.
  • Reset reloads the original array seed, while New Array spins up fresh randomized data.
  • The Nested Loop Test is a debug baseline to highlight the render cadence of the canvas.
